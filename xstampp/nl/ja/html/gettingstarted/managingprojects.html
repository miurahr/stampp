プロジェクト管理

プロジェクト作成

新規プロジェクトを作成する二つの方法があります。ようこそ画面では、をクリックすると、新規プロジェクトを作成できます。それ以外では、メニューバーのファイルを選択し、新規プロジェクト作成をクリックしてください。

プロジェクト保存

メニューバーのファイル -> プロジェクト保存 を選択することで、既存プロジェクトの変更点を保存できます。When saving for the first time, you will be asked to enter a file destination and a file name.

プロジェクトのエキスポート

PDFエキスポートを使うと、ドキュメントを簡単に共有できます。メニューバーのファイルから、.名前を付けてエキスポートをクリックします。. エキスポートのパラメータ変更のための小さなウインドウが表示されます。
  • 会社PDFエキスポートファイルの表紙に表示される会社名を入れることができます。
  • ロゴPDFエキスポートファイルの表紙に表示される会社のロゴや任意のイメージを定義できます。
  • 背景とフォント色カスタムのページデザインとして色を変更できます。
  • 宛先PDFファイルの予定している宛先をここに設定できます。
