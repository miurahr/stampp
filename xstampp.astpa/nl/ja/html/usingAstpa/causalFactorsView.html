Using the Causal Factors Table

Adding a Causal Factor

After finishing the Control Structure, each Node of the Control Structure can be used in the Causal Factors Table to add a Causal Factor to it. Press in the column labeled &quotCausal Factor" to do so.

Link a Hazard to a Causal Factor

Causal Factors can be linked to Hazards. Press in the column labeled &quotHazard Links" to do so. It&acutes the same linking procedure as described for the Unsafe Control Actions Table.

Define or Link Safety Constraint corresponding to a Causal Factor

New Safety Constraints can be defined for Causal Factors as they can be for Unsafe Control Actions. To do so, press in the column labeled as &quotSafety Constraint". If this Safety Constraint was already defined in the Corresponding Safety Constraints Table, it should not be defined one more time. Press to link to existing Safety Constraints.
