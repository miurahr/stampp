Editing the Proximal Events

You can add an event by clicking on the following button . If you want to delete an event you have to select it in the table an then press the button.

Editing the Table

If you want to edit a column in the table just make a doubleClick on it and enter a value. The table has the following columns: ID (unique id for the event), Date(the date of the event with a dd.mm.yyyy format), Time(the time of an event with format hh.mm.ss), Description (here you enter a Description of the event) .
